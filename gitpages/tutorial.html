Publicando no GitHub Pages: Um Tutorial Simples

Passo 1: Crie um Repositório

Crie um repositório no GitHub para o seu projeto. Certifique-se de ter um arquivo index.html na raiz ou em um diretório chamado docs ou public.

Passo 2: Estrutura do Repositório

Verifique se a estrutura do seu repositório está adequada para um site. Por exemplo:

.
|-- index.html
|-- style.css
|-- script.js

Passo 3: Ative o GitHub Pages

Vá para as configurações do seu repositório, vá até a seção "GitHub Pages" e escolha o branch desejado. Clique em "Save".

Passo 4: Acesse seu Site

Após ativar o GitHub Pages, seu site estará disponível em:

https://seunomeusuario.github.io/seurepositorio

Passo 5: Atualize Seu Site

Cada vez que você fizer push para o branch selecionado, seu site será atualizado automaticamente.

Passo 6: Configurando um Domínio Personalizado

Se você possui um domínio próprio, pode configurá-lo para apontar para o seu site no GitHub Pages. Aqui está como:

Subpasso 6.1: Compre um Domínio

Adquira um domínio de um registrador de domínios de sua escolha, como GoDaddy, Namecheap ou Google Domains.

Subpasso 6.2: Configurando o Domínio no GitHub

  1. No seu repositório do GitHub, vá para as configurações.
  2. Na seção "GitHub Pages", adicione o seu domínio personalizado no campo "Custom domain".
  3. GitHub Pages gerará automaticamente um arquivo chamado CNAME no seu repositório. Certifique-se de que esse arquivo está presente.

Subpasso 6.3: Configurando as Configurações DNS

Acesse o painel de controle do seu registrador de domínios e configure as configurações DNS. Crie registros CNAME apontando para seunomeusuario.github.io.


    Tipo: CNAME
    Nome: www (ou o subdomínio desejado)
    Valor: seunomeusuario.github.io

Subpasso 6.4: Aguarde a Propagação DNS

Após configurar as configurações DNS, pode levar algum tempo para que as alterações se propaguem pela internet. Isso pode levar algumas horas até 48 horas.

Passo 7: Acesse Seu Site com o Domínio Personalizado

Depois que a propagação DNS estiver completa, seu site estará disponível em seu domínio personalizado, por exemplo, www.seudominio.com.

Considerações Finais

Parabéns! Agora, seu site no GitHub Pages está associado ao seu próprio domínio.
